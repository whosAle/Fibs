The following program computes a sequence of fibonacci numbers and displays it using MARS (an interactive development enviorment for programming in MIPS assembly langauge).

Input: Reads three input values from the keyboard:
	1) the first number in sequence
	2) the second number in the sequence
	3) number of elements of the sequence

Output: 
	1) number in decimal notation
	2) number oin hexadecimal
	3) number of 1-bits in the binary represenation of the number

Example:
3 	0x00000003	2
1 	0x00000001	1
4 	0x00000004	1
5 	0x00000005	2
